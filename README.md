# Marco Portaro — Sito personale

Landing page personale in stile Apple (font di sistema SF Pro, palette minimale,
tipografia grande, micro-animazioni al reveal).

## Struttura

- `index.html` — markup della pagina (hero, chi sono, progetti, filosofia, contatti)
- `css/styles.css` — stile, dark mode automatica, responsive
- `js/main.js` — nav sticky, progress bar, reveal on scroll, menu mobile

## Sviluppo locale

Nessuna build necessaria: apri `index.html` in un browser, oppure servi la cartella
con un server statico, ad esempio:

```bash
python3 -m http.server 8000
```
